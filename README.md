# Sightseeing

🗺️ Sightseeing - 관광지 조회 서비스

## 📌 기능 소개
Sightseeing 모듈은 전체, 수도권, 충청권, 전라권, 경상권, 강원권, 제주권의 관광지를 리스트로 조회하고, 각 관광지의 상세 정보를 확인할 수 있는 기능을 제공합니다.
- 키워드 기반 관광지 검색
- 관광지 목록 페이지네이션
- 관광지 상세 정보 보기
- 상세 페이지에서 관련 관광지 추천


## 💻 기술 스택
Frontend: HTML, CSS, JavaScript
Backend: Spring Boot
DB: MySQL
API 통신: RESTful API (JSON)
